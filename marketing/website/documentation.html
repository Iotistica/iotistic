Iotistic Agent

A lightweight container orchestrator for edge devices. Deploy and manage Docker containers or Kubernetes workloads on Raspberry Pi and other edge hardware.

Quick Installation

curl -sfL https://apps.iotistic.ca/agent/install | sh

Features

  • ✓ Container orchestration (Docker & K3s)
  • ✓ Remote updates and management
  • ✓ Built-in VPN connectivity
  • ✓ Multi-architecture support (ARM64, ARM32, x86-64)
  • ✓ Real-time monitoring and metrics
